требования к вашим работам: 
 
1+  - работа должна выглядеть фактически 1 в 1 как на задании, с нюансами, которые описаны ниже  
 
2+  - создаете загрузочный экран, в стилистике выбранного сайта, с любым эффектом открытия через 3-5 секунд, пока виден экран загрузки - никакой полосы прокрутки не должно быть 
 
3+  - во всех работах, первая секция - размером в экран(в работе aweom1 это первая секция + шапка над ней идут размеров в экран) 
 
4+  - кнопки вверх/вниз для плавной прокрутки страницы на самый верх или низ, кнопки всегда на экране  
 
5+  - полная адаптивность сайта при максимальном сжатии экрана. На мобильных устройствах все должно быть отлично во всех положениях экрана (портретный, альбомный). Не просто уменьшино, а именно перестроина структура под текущий размер экрана  
 
6+  - у всех изображений есть эффект наведения (затемнение, размытие, повороты и повление внутри текста, иконок и т.д.)  
 
7+  - все выполняется на чистом html\css\svg 
 
8+  - структура, цветовая гамма, шрифты, все элементы и оформление должны полностью совпадать с выбранным шаблоном (1 в 1) (дальше будут уточнения)  
 
9+  - у каждой кнопки есть эффект наведения 
 
10+  - у поля для ввода есть эффект наведения и фокусировки  
  
11+  - если где-то показан какой-то эффект у вас должно быть точно такой  (цвет текст другой, черта есть, рамка, иконки и цвет внутри картинки,градиент и т.д.)   
 
12+  - графику вы подбираете максимально подходящую к тому, что у вас в шаблоне ( если там фото человека - вы подбираете фото человека, если у вас там иконка настроек - у вас тоже должна быть иконка настроек, но не обязатольно такая же как там ) 
 
13+ - кнопки плеера для видео и т.д. должны быть точно такие же как и в примере 
 
14+     - кнопка плеера для видео не запускает видео!, она только как декорация с эффектом hover  
15 + - если на сайте есть видео - у вас так же должно быть подключено видео, музыка и т.д., но без встроеных элементо вуправления(вы их не показываете)  
 
16+  - если есть карта - подключаете карту 
 
17+  - если есть кнопка скрытого меню и т.д. ( квадрат с тремя полосками и прочее ), значит - по клику на нее должно появляются меню или панель, любая  
 
18+  - под мобильную версию обязательно бургер меню ( все меню прячете и показываете кнопку, по которой покажется мобильное меню ) 
 
19+  - по клику на пункты меню - страница должна прокручиватся к определенной секции( какой пункт на какую секцию ведет - сами рещаете, должны идти подряд) 
 
20+  - вы можете использовать любые фишки Html и css, но если вы не сможете ответить на вопросы по коду - это понизит оценку  
 
21-  - все кнопки являются ссылками, либо якорными либо никуда не ведут, но обязательно ссылками ( должен обязательно быть атрибут href, но он может быть пустым) 
 
22+  - если текст идет особым шрифтом (не стандартным) -  у вас тоже должен использоваться максимально похожий шрифт  
 
23+  - в обязательном порядке используйте переменные, но только в тех местах, где они нужны 
   
24+  - должны быть указаны мета данные : автор, в чем написан сайт, его версия, ключевые слова, описание, иконка и название сайта, язык 
 
25+  - работа полностью должны проходить валидацию(убрать все ошибки, warn-ы можно оставить, если сможете пояснить их причину) 
 
 26+ - скроллбар должен быть кастомным, в стиле сайта ( в firefox может не работать - это приемлемо)  
 
 27+ - обязательно использовать вендарные префиксы для слабо поддерживаемых, не стандартных или новых свойств ( которые недавно добавлены )  
 
 28- - работа должна быть выгружена на бесплатный хостинг\хостинг вашего github репозитория 
 
 29+ - в работе должен быть кастомный курсор, подходящий по стилю и цветовой гамме к работе  
 
 30+ - все подключаемые файлы должны лежать на хостинге, вместе с файлом документа, подключать по ссылке из интернета - нельзя. Так же файлы должны быть разбиты на директории, не валятся всем скопом в коне проекта 
 
 31- - правильно разделите стили между css файлами, 1н файл недопустим 
 
 32+ - если есть слайдер или боковые стрелки: 
   33+ - делаете смену слайдов не по клику, а через анимацию(меняются сами, клики ничего не делают)

34- - если хотите - можете сделать чтобы по клику переключались слайды(это сложно но будет вам в плюс) 
 
 35+ - цвет выделения текста в проекте не должен быть стандартным и должен подходить по цвета макета  
 
 36+ - структура сайта должна состоять из section, header, footer, main, а не только из div 
  37+ - весь текст можно заменить на рыбу(lorem ipsum...) 
 
 38+ - можно использовать любой материал, если сможете ответить по нему на вопросы
